# 📚 Book Tracker - Трекер прочитанных книг

**Автор:** Даниил Ильин

## 📖 Описание проекта

**Book Tracker** — это графическое приложение для управления списком прочитанных книг. Приложение позволяет добавлять, редактировать, удалять и фильтровать книги, а также просматривать статистику по вашей библиотеке. Все данные автоматически сохраняются в файл JSON.

## ✨ Функциональные возможности

### Основные функции
- **Добавление книг** с указанием названия, автора, жанра и количества страниц
- **Редактирование** существующих записей
- **Удаление книг** из коллекции
- **Автоматическое сохранение** данных при каждом изменении
- **Ручное сохранение** по кнопке "Сохранить вручную"
- **Просмотр статистики** по библиотеке

### Фильтрация
- **По жанру** - выбор конкретного жанра из выпадающего списка
- **По количеству страниц** - фильтрация с операторами "больше", "меньше", "равно"

### Валидация данных
- **Название книги**:
  - Не может быть пустым
  - Максимальная длина - 200 символов
- **Автор**:
  - Не может быть пустым
  - Максимальная длина - 100 символов
- **Жанр**:
  - Обязательный выбор из списка
- **Количество страниц**:
  - Должно быть целым положительным числом
  - Диапазон: от 1 до 10000

### Хранение данных
- Все данные сохраняются в файл `books.json`
- При запуске приложение автоматически загружает сохраненную коллекцию
- Формат JSON обеспечивает совместимость с другими программами
